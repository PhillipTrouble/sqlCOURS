MySQL

    MySQL peut s'utiliser en ligne de commande (Bash ou CMDR) ou avec une interface graphique(php My Admin)
    Pour se connecter à MySQL en ligne de commande on utilise mysql -u utilisateur -p

        + sur la ligne de cmd

        En résumé

        • MySQL est un SGBR Système de Gestion de base de données Relationnelles
        • le Langage SQL est utiliser pour la communication entre le client et le serveur
        • Dans une BDD les données sont représentées sous forme de tables
        • Dans une table nous avons des colonnes , chaque données sera introduite sous forme de ligne dans la table

        View details »

        SGBDR

        • Un système qui permet de gérer une BDD /// CRUD create, read, update, delete
        • Le logiciel client permet de dialogner avec le serveur avec le langage SQL
        • MySQL Structured Query Langage
        • Date de naissance 1994
        • Open source
        • Concurrents : Oracle, MS Acces SQLite PostgreSQL

        Requêtes de base, création

        • Créer une base CREATE DATA BASE basetest CHARACTER SET 'utf8';
        • Supprimer une BDD DROP DATABASE IF EXISTS basetest
        • Créer une table : CREATE TABLE t_utlisateurs
        • Insérer dans une table : INSERT INTO t_utilisateur
        • Afficher le contenu d'une table SELECT * FROM t_utilisateur
